Installation
------------

To install phir, simply run:

.. code-block:: bash

   pip install phir

Usage
-----

To use phir, import the main module:

.. code-block:: python

    from phir import main

    main.run()

Utils
-----

The utils module contains various utility functions that can be used in conjunction with phir. To use the utils module, import it like so:

.. code-block:: python

    from phir import utils

    utils.do_something()
